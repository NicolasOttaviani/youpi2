# youpi2

Little football game.

## Getting started

### Install the project

```bash
git clone https://github.com/NicolasOttaviani/youpi2.git
yarn install
```

### Running the project

```bash
yarn run dev
```

Open up [localhost:3000](http://localhost:3000) and start clicking around.

### Running test

```bash
yarn run test
```

Open up [localhost:3000](http://localhost:3000) and start clicking around.
